# 수들의 합 4

## 난이도

- 골드 4

## 문제

https://www.acmicpc.net/problem/2015

## 알고리즘 분류

- 자료 구조
- **누적 합**
- **해시를 사용한 집합과 맵**
- 트리를 사용한 집합과 맵

## 회고



```c++
#include <bits/stdc++.h>

typedef long long ll;
using namespace std;

int main() {
    int N, K;
    cin >> N >> K;
    vector<int> prefixSum(N + 1, 0);
    unordered_map<ll, ll> map;
    for (int i = 1; i <= N; i++) {
        int num;
        cin >> num;
        prefixSum[i] = prefixSum[i - 1] + num;
    }
    ll ans = 0;
    for (int i = 1; i <= N; i++) {
        if (prefixSum[i] == K) {
            ans++;
        }
        ans += map[prefixSum[i] - K];
        map[prefixSum[i]]++;
    }
    cout << ans;
}
```
